# Course Objectives

Students will:

- Model relative motion with Clohessy-Wiltshire dynamics.
- Generate impulsive rendezvous maneuvers.
- Implement operational constraints.
- Build an offline planning graph.
- Develop an A* mission planner.
- Demonstrate autonomous maneuver planning in MATLAB.
